Use the highlight attribute to emphasize specific parts of the HTTP exchange

1. Highlight Response Status Line

Perfect for drawing attention to the response status:
<http-console highlight="response-line"></http-console>

2. Highlight Response Headers

Focus on the response headers:
<http-console highlight="response-headers"></http-console>

3. Highlight Response Body

Emphasize the response body content:
<http-console highlight="response-body"></http-console>

4. Highlight Multiple Sections

You can highlight multiple sections using comma-separated values:
<http-console highlight="response-line,response-headers"></http-console>

5. Highlight Request Sections

Works with request sections too:
<http-console highlight="request-line,request-body"></http-console>

6. Full Example - Multiple Highlights

Highlight both request and response sections:
<http-console highlight="request-headers,response-line,response-body"></http-console>

7. Highlight Specific Request Header

Highlight only the Content-Type request header:
<http-console highlight="request-header:Content-Type"></http-console>

8. Highlight Specific Response Header

Highlight only the Location response header:
<http-console highlight="response-header:Location"></http-console>

9. Highlight Multiple Specific Headers

Highlight Authorization request header and Content-Type response header:
<http-console highlight="request-header:Authorization,response-header:Content-Type"></http-console>

10. Mix Sections and Specific Headers

Combine section and specific header highlighting:
<http-console highlight="response-line,request-header:Content-Type,response-header:Location"></http-console>
