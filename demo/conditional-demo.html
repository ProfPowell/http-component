Conditional Rendering Demo

The http-console component now only shows sections when data is provided. If you omit request or response data, that section won't be displayed.

1. Request Only

Only request data provided - response section is hidden:
console.data = { request: {...} }

2. Response Only

Only response data provided - request section is hidden:
console.data = { response: {...} }

3. Both Request and Response

Both request and response data provided (traditional behavior):
console.data = { request: {...}, response: {...} }

4. No Data

No data provided - shows placeholder message:
console.data = {}
