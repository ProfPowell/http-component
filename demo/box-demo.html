Box Feature Demo

The box attribute allows you to add visual borders with labels around different sections of HTTP exchanges. This is useful for educational purposes, documentation, and drawing attention to specific parts of the exchange.

Syntax: Same as highlight attribute

  • box="response-line" - Box the response status line
  • box="request-headers" - Box all request headers together with a label
  • box="response-header:Content-Type" - Box a specific header
  • box="request-line,request-headers,request-body" - Box multiple sections

1. Box Response Line

Draw a border around just the response status line:
<http-console box="response-line"></http-console>

2. Box All Request Headers

Box all request headers together with a "Request Headers" label:
<http-console box="request-headers"></http-console>

3. Box Response Body

Box the response body to emphasize the payload:
<http-console box="response-body"></http-console>

4. Box Specific Headers

Box specific individual headers with their names as labels:
<http-console box="request-header:Authorization,response-header:Location"></http-console>

5. Box All Three Request Sections

Box request line, headers, and body to show the complete request structure:
<http-console box="request-line,request-headers,request-body"></http-console>

6. Box All Three Response Sections

Box response line, headers, and body to show the complete response structure:
<http-console box="response-line,response-headers,response-body"></http-console>

7. Combine Box and Highlight

You can use both box and highlight together:
<http-console box="response-line,response-body" highlight="response-header:Content-Type"></http-console>

8. Mixed: Box Sections and Individual Headers

Box the response line and specific headers:
<http-console box="response-line,response-header:Content-Type,response-header:Location"></http-console>
